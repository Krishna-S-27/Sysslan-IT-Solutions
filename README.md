# Sysslan IT Solutions - Java Practice Roadmap

This repository is a structured collection of Java projects built level by level, starting from core fundamentals and moving toward more professional mini applications. It is designed as a learning roadmap as well as a showcase of progressive Java practice.

## What This Repository Contains

- Beginner-friendly console programs
- Intermediate Java utilities
- Object-oriented mini projects
- Real-world beginner applications
- Advanced beginner projects with multi-class structure, file handling, configuration, and reporting

## How To Use

1. Start from Level 1 and move upward in order.
2. Open the `README.md` inside each level folder for task details.
3. Compile and run each project from its own folder.
4. Use the higher levels to study cleaner Java project structure and more practical program design.

## Run Basics

For simple single-file Java programs:

```powershell
javac FileName.java
java ClassName
```

For small multi-file Java projects:

```powershell
javac *.java
java MainClassName
```

For Level 5 structured projects, use the project-specific commands described in the Level 5 README and each project folder README.

## Purpose

This repository is meant to help with:

- strengthening Java fundamentals
- practicing object-oriented programming
- learning file handling and data processing
- building portfolio-ready Java mini projects

It can be used both as a personal learning archive and as a clean progression-based Java project collection.
