# Level 4 - Real-World Beginner Projects

This folder contains Level 4 tasks that feel closer to practical applications. These projects combine object-oriented design, file handling, reporting, and data processing.

## Task 1: CSV Data Processor
- Folder: `CSVDataProcessor`
- Files: `CSVDataProcessor.java`, `CSVReader.java`, `DataAnalytics.java`
- Purpose: Reads CSV data and performs basic analysis or reporting.

## Task 2: Task Manager
- Folder: `TaskManager`
- Files: `TaskManager.java`, `Task.java`
- Purpose: Manages tasks with operations such as add, track, and update.

## Task 3: Student Report Generator
- Folder: `StudentReportGenerator`
- Files: `StudentReportGenerator.java`, `Student.java`, `Course.java`
- Purpose: Builds student-related reports from structured course and record data.

## Task 4: File Organizer
- Folder: `FileOrganizer`
- Files: `FileOrganizer.java`, `FileInfo.java`, `FileType.java`
- Purpose: Categorizes and organizes files based on type and file metadata.

## How To Run

Compile all Java files inside the chosen project folder, then run the main class.

Example:

```powershell
cd ".\FileOrganizer"
javac *.java
java FileOrganizer
```
