# Level 2 - Intermediate Java Utilities

This folder contains Level 2 tasks that move beyond basics into practical console utilities. These projects use conditionals, loops, strings, file handling, and slightly more structured program logic.

## Task 1: Expense Tracker
- Folder: `ExpenseTracker`
- File: `ExpenseTracker.java`
- Purpose: Records and manages expenses with a simple Java-based tracking workflow.

## Task 2: File Encryption
- Folder: `FileEncryption`
- File: `FileEncryption.java`
- Purpose: Demonstrates file processing and basic encryption or transformation logic.

## Task 3: Password Strength Analyzer
- Folder: `PasswordStrengthAnalyzer`
- File: `PasswordStrengthAnalyzer.java`
- Purpose: Evaluates password quality using rules such as length, symbols, and character variety.

## Task 4: Tic Tac Toe
- Folder: `TicTacToe`
- File: `TicTacToe.java`
- Purpose: A console-based game project using arrays, conditions, and game flow handling.

## How To Run

Open PowerShell in the required task folder and run:

```powershell
javac FileName.java
java ClassName
```

Example:

```powershell
cd ".\TicTacToe"
javac TicTacToe.java
java TicTacToe
```
