# Level 1 - Core Java Fundamentals

This folder contains the foundational Java tasks for Level 1. These programs focus on basic syntax, console input/output, loops, conditions, and simple problem solving.

## Task 1: Temperature Converter
- Folder: `temperatureConverter`
- File: `TemperatureConverter.java`
- Purpose: Converts temperatures between common units through a simple Java console program.

## Task 2: Student Grade Calculator
- Folder: `StudentGradeCalculator`
- File: `StudentGradeCalculator.java`
- Purpose: Accepts marks, calculates totals or averages, and prints a grade summary.

## Task 3: Password Generator
- Folder: `PasswordGenerator`
- File: `PasswordGenerator.java`
- Purpose: Generates passwords using Java logic and basic string handling.

## Task 4: Number Grid
- Folder: `NumberGrid`
- File: `NumberGrid.java`
- Purpose: Prints a number pattern or grid using loops and formatted output.

## How To Run

Open PowerShell in the required task folder and run:

```powershell
javac FileName.java
java ClassName
```

Example:

```powershell
cd ".\PasswordGenerator"
javac PasswordGenerator.java
java PasswordGenerator
```
