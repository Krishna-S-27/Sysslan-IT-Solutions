# Level 3 - OOP & Small Projects

This folder contains Level 3 tasks that introduce object-oriented programming through small but realistic Java projects. These tasks use classes, objects, encapsulation, and multi-file project structure.

## Task 1: Contact Manager
- Folder: `ContactManager`
- Files: `ContactManager.java`, `Contact.java`
- Purpose: Stores and manages contact details using custom Java classes.

## Task 2: Library Management System
- Folder: `LibraryManagementSystem`
- Files: `LibraryManagementSystem.java`, `Book.java`, `Member.java`
- Purpose: Simulates a small library system with books, members, and basic management operations.

## Task 3: Student Record System
- Folder: `StudentRecordSystem`
- Files: `StudentRecordSystem.java`, `Student.java`
- Purpose: Manages student data using object-oriented design.

## Task 4: Timestamped Log
- Folder: `TimestampedLog`
- Files: `Logger.java`, `LogEntry.java`, `LogLevel.java`
- Purpose: Builds a structured logging system with log entries and severity levels.

## How To Run

Compile all Java files inside the chosen project folder, then run the main class.

Example:

```powershell
cd ".\LibraryManagementSystem"
javac *.java
java LibraryManagementSystem
```
